Simple React App

Trabalho individual da disciplina de Desenvolvimento Web com React da Residência em TIC do Serratec – Parque Tecnológico da Região Serrana.

O objetivo deste projeto é aprender sobre componentes, props e rotas em aplicações React, explorando a estrutura e funcionamento básico de um projeto front-end moderno.

Como executar o programa
# Clone este repositório
git clone https://github.com/seuusuario/simple-react-app.git

# Acesse a pasta do projeto
cd simple-react-app

# Instale as dependências
npm install

# Execute o servidor de desenvolvimento
npm run dev

O projeto será iniciado e poderá ser acessado pelo navegador no endereço exibido no terminal (geralmente http://localhost:5173).
